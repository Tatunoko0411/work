├ 02_開発環境\
│├ 01_Client\           # Unityのプロジェクト一式の中から以下のフォルダを除いたもの
││                      #   /Library
││                      #   /obj
││                      #   /build
││                      #   /Logs
││                      #   /Temp
││
│├ 02_Server\           # 
││├ 01_API\            # APIのプロジェクト一式（Debug, Releaseフォルダは削除する)
││└ 02_DB\             # PHPStormで作成したSQLファイル (関連データベース、テーブルをcreateするためのクエリ。コメントで説明をつけておく)
│└ [開発環境説明資料]   # クライアント(Unity) ⇔ サーバー(AzureFunctions) ⇔ データベース(AzureDBforMySQL)を図や説明文で作成

│
├ 03_動画\               # 1分程度でまとめる (ゲームプレイ動画。サーバー通信の説明も入れるとベスト！)
├ 04_スクリーンショット\ # 何枚かいれておく
└ 05_アピールシート\     # パワポ1ページ程度でゲームやアピールポイントの紹介
